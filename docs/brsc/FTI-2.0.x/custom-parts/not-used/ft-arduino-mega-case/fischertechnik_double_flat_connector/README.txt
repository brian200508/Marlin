# Summary

A variant of the standard [fischertechnik](https://en.wikipedia.org/wiki/Fischertechnik) [connector](https://ft-datenbank.de/ticket/4510) with two flat sides in three versions with 15mm, 30mm, or 45mm. It is meant for situations when the ft [spring cam (Federnocke)](https://ft-datenbank.de/ticket/1387) will not provide enough stability.
