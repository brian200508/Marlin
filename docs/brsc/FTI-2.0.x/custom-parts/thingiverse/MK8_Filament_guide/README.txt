# Summary

Help to guide the filament into the MK8 extruder, the beauty of this part you don't need to undoing tension screw, it clamp around the end of the block.
Heat hot end, remove filament once hot enough, remove the Cooling fan and insert this part.
Once Cooling fan back in place, add filament again.
