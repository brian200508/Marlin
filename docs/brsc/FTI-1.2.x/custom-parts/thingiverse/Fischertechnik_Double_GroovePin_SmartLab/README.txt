# Summary

These are 15 and 30 mm versions of what I call a double groove connector. My students lose these so it is helpful to be able to print them. I did these in ABS on a da Vinci 1.0 AiO with glue stick on the bed, no raft or brim and no supports, using the "Good" setting (not quite the top resolution). 10% was the default fill setting, but this is inconsequential on such a small part. 

This thing was made with Tinkercad. Edit it online https://www.tinkercad.com/things/7xtlIzkcgIN
